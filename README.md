# userAuth

## Microserviço de autenticação 

Projeto desenvolvido durante o Bootcamp Eduzz Fullstack Developer na [DIO](https://digitalinnovation.one/)


### Tecnologias utilizadas

* Nodejs
* Typescript

### Contém Endpoints Base

#### Usuários

* GET /users
* GET /users/:uuid
* POST /users
* PUT /users/:uuid
* DELETE /users/:uuid

#### Autenticação

* POST /token
* POST /token/validate
